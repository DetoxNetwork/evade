Evade is a giant collection of bookmarklets to help you or just keep you entertained when there's nothing to do. What are bookmarklets, you may ask? They're snippets of JavaScript code that are put into a bookmark, allowing you to do anything from playing tic tac toe to opening a search engine in a tab. To use them, click on a button, and if you like how that one sounds you can right click the link that says "code" and copy the link address. Once you've done this, bookmark the page you're on and change the title to whatever you like. Then, hit edit and you can paste the code into the URL tab. Otherwise, if you just want to test them first, click the code button and they will run on the current page.
